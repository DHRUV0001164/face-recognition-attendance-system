# Face Recognition Attendance System

## Overview
Real-time Face Recognition Attendance System using Python, OpenCV, Streamlit, and face-recognition library.
Detects and recognizes faces from webcam and saves attendance automatically.

## Features
- Live webcam face detection
- Register new students with course and year
- Automatic attendance saving in CSV
- Streamlit-based web interface

## Tech Stack
- Python
- OpenCV
- face-recognition (dlib)
- Streamlit
- NumPy & Pandas
